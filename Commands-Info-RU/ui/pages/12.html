Прокси-модуль TERA: Item-id-finder.

Описание:

▼ Модуль, который показывает данные, связанные с предметом, после наведения на него курсором.

Использование:

▼ Чтобы увидеть результаты в файле сохранения, вам нужно снова отключить эту функцию или закрыть игру.
▼ Цифры в скобках файла сохранения указывают время создания.
▼ Файл сохранения находится в папке модулей в разделе «Дополнительные данные».

Команды:

•finditem message ► Чтобы показать данные найденного элемента в чате панели инструментов. По умолчанию отключено.
•finditem logger ► Сохранить данные найденного элемента в файл сохранения. По умолчанию отключено.
•finditem getall ► Чтобы сохранить все доступные данные элемента в файл сохранения.
•finditem config ► Чтобы включить или отключить функции, написанные выше.
