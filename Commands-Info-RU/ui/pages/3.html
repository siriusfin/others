Прокси-модуль TERA для автоматического использывания зелья HP MP.

Описание:

▼ Поддержка режима убийства.
▼ Поддержка зелья на поле боя.
▼ Перезагрузка файла с помощью команды, без необходимости перезапуска или перезагрузки.
▼ Требуется прокси Caali "tera-game-state".

Команды:

•pot ► Для включения модуля.
• pot id [item link] ► получить идентификатор элемента <удержание Ctrl + левый клик>для элемента link.
•pot hp ► включение и отключение автоматического использования hp.
•pot mp ► включение и отключение автоматического использования mp.
•pot slaying ► Включение и отключение автоматического использования hp зелья с режимом убийства.
•pot reload hp ► Перезагрузить файл HP.json * открыть инвентарь для обновления предмета.
•pot reload mp ► Перезагрузить файл MP.json * открыть инвентарь для обновления предмета.
•pot reload config ► Перезагрузить файл Config.json.
•pot debug ► Show DEBUG.
