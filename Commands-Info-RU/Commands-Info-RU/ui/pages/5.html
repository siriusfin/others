Прокси-модуль Detector для обнаружения игроков.

Описание:

▼ Уведомляет вас, когда[Player]находится рядом, даже если они [невидимы].
▼ Будет виден луч света (шлейф), следующий за целью (движение / бой / полет).

Команды:

•поиск ► Для включения модуля.

Примечание:

• Отредактируйте config.json, чтобы добавить соответствующий ник игрока.
